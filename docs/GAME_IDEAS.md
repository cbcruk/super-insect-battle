# 슈퍼곤충대전 - 게임 아이디어

## 역사적 배경

곤충 싸움은 아시아 전역에서 수천 년간 이어져 온 전통 문화이다.

### 중국 - 귀뚜라미 싸움

- 당나라(618~907)부터 2000년 역사
- 황제의 취미에서 시작 → 서민 문화로 확산
- 남송 재상 가사도(賈似道)가 귀뚜라미 싸움 가이드북 저술
- 현대: 전국 대회 개최, 산동성 닝양현은 연 940억원 규모 시장
- 각 귀뚜라미는 개별 토기 항아리에 보관
- 식단: 새우, 팥, 간, 구더기 등으로 컨디션 관리

### 일본 - 장수풍뎅이/사슴벌레

- 카부토무시(兜虫) = 장수풍뎅이, 사무라이 투구에서 유래
- 쿠와가타무시(鍬形虫) = 사슴벌레, 투구 장식에서 유래
- 17세기부터 곤충 판매가 전문 직업화
- 여름 축제(오봉)와 연계된 문화

### 동남아 - 베트남, 태국

- 지역 축제와 연계
- 시골 지역에서 브리더, 포획꾼, 판매상 등 생계 수단

---

## 핵심 컨셉: 불확실성과 도박

실제 곤충 싸움의 매력은 **결과를 예측할 수 없다**는 점이다.
이 불확실성이 도박 요소를 만들고, 사람들을 열광하게 한다.

### 현재 문제점

```
장수풍뎅이 vs 사슴벌레 = 99.2% 승률
→ 스탯만 보면 결과가 뻔함
→ 도박의 재미가 없음
```

---

## 불확실성 시스템 아이디어

### 1. 숨겨진 정보

- 상대 곤충 스탯이 **보이지 않음** (외형으로만 추측)
- IV(개체값)는 본인도 정확히 모름 (대략적 등급만 표시: S/A/B/C)
- "이 귀뚜라미 좋아 보이는데..." → 안목이 곧 실력
- 배틀 경험이 쌓이면 상대 곤충을 더 정확히 평가 가능

### 2. 컨디션 시스템

```typescript
interface BattleCondition {
  // 매 배틀마다 0.7 ~ 1.3 범위의 컨디션 적용
  physical: number // 물리 스탯에 영향
  mental: number // 명중률, 회피율에 영향
  stamina: number // 후반 턴 성능에 영향
}
```

- 같은 곤충도 컨디션에 따라 결과가 달라짐
- 좋은 날: 약한 곤충이 강한 곤충을 뒤집음
- 컨디션 관리가 육성의 핵심

### 3. 배당률/베팅 시스템

```
[오늘의 대전]
🪲 장수풍뎅이 (1.2배) vs 🦗 사슴벌레 (3.5배)
      ↑ 유리해 보임           ↑ 불리해 보이지만...

베팅 금액: 1000골드
→ 장수풍뎅이 승리 시: 1200골드 획득
→ 사슴벌레 승리 시: 3500골드 획득
```

- 강해 보이는 쪽 = 낮은 배당
- 언더독 = 높은 배당, 대박 가능성
- NPC 배틀 관전 + 베팅

### 4. 심리전 요소

- **블러핑**: 일부러 약한 곤충을 먼저 내보내는 척
- **픽/밴**: 토너먼트에서 상대 강한 곤충 금지
- **패턴 읽기**: 상대 플레이어의 선호 스킬/전략 분석

## 지역 시스템

### 곤충 서식지

| 지역            | 특화          | 보너스             |
| --------------- | ------------- | ------------------ |
| 산동성 (중국)   | 귀뚜라미      | 공격력 +10%        |
| 닝양현 (중국)   | 레어 귀뚜라미 | IV 상한 증가       |
| 니가타현 (일본) | 장수풍뎅이    | 방어력 +10%        |
| 오키나와 (일본) | 사슴벌레      | 속도 +10%          |
| 메콩강 유역     | 전갈, 거미    | 상태이상 확률 +15% |

### 지역 대회

- 각 지역 예선 → 전국 대회 → 세계 대회
- 지역별 특수 룰 (예: 산동성은 귀뚜라미만 참가)

## 육성 시스템

### 식단 관리

| 먹이         | 효과             |
| ------------ | ---------------- |
| 새우 분말    | ATK 성장 촉진    |
| 팥           | HP 성장 촉진     |
| 간 (양/염소) | SP.ATK 성장 촉진 |
| 고단백 사료  | 전체 성장 (느림) |

### 컨디션 관리

- 휴식: 컨디션 회복
- 훈련: 스탯 성장 but 컨디션 소모
- 대회 전 암컷 투입: 투지 상승 (ATK 버프)

### 장비 - 항아리/토기

| 항아리        | 효과                                  |
| ------------- | ------------------------------------- |
| 일반 토기     | 기본 휴식 효율                        |
| 고급 도자기   | 휴식 효율 +20%                        |
| 골동품 항아리 | 휴식 효율 +50%, 레어 컨디션 확률 증가 |

## 수집 요소

### 희귀도 시스템

- **일반**: 아무 곳에서나 포획
- **레어**: 특정 지역/시즌에만 출현
- **전설**: 스토리 진행으로 획득, 역사 속 명충(名蟲) 재현

### 혈통/계보

- 부모 개체의 IV가 자식에게 영향
- 명문 혈통 = 높은 IV 확률
- 브리딩 시스템
